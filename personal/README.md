# Assets for this site

1. **`profile.png`** — Sidebar photo (in the repo).
2. **`academic_cv.pdf`** — Curriculum vitae PDF (in the repo; update this file when your CV changes).

The site links to `personal/academic_cv.pdf` from the **Home** and **CV** pages.

**Optional — Outside academia:** you can add `outside-1.jpg` / `outside-2.jpg` (or `.png`) and uncomment the gallery block in `outside.html` (see the comment in that file). Keep files reasonably small for fast loading.
